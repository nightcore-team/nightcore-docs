# Компоненты системы кланов

---

## ClanShopView
!!! info ""
    Информация о покупке:

    - Пользователь (упоминание и id)
    - Клан (название и id роли)
    - Баланс клана до покупки
    - Баланс клана после покупки
    - Название товара
    - Цена товара
    - Идентификатор покупки - id созданной ветки

    ### Доступ:
    CLANS_ACCESS

    ### Необходимые права бота:
    - send_messages - доступ к отправке сообщений в канал для логирования клановой системы (в случае если он настроен), обязательно для отправки лога одобрения или отклонения покупки
    - manage_threads - для закрытия и архивирования ветки

!!! note "Информация в логе"
    - `Клан` - название клана, в котором было совершено действие
    - `Покупатель` - пользователь, купивший товар (Упоминание + id)
    - `Предмет` - название предмета
    - `Цена` - цена предмета, в репутации
    - `Баланс клана до покупки` - количество репутации клана до покупки товара
    - `Баланс клана после покупки` - количество репутации клана после покупки товара
    - `Модератор` - пользователь, отклонивший или принявший покупку товара (Упоминание и id)
    - `Статус заказа` - approved - товар одобрен для покупки, denied - отклонен
