# Компоненты системы тикетов

---

## CreateTicketButton
!!! info ""
    Компонент прикрепляется к сообщению в команде `/ticketmessage`

    Используется для создания тикета

    ### Необходимые права бота
    - `manage_channels` - доступ к управлению каналами для создания нового канала тикета

!!! warning "Возможные ошибки"
    - Конфигурация тикетов не настроена, в случае если не настроен хотя бы один из параметров:
        - Роль, которая пингуется при создании тикета
        - Категория для новых тикетов
        - Категория для закрепленных тикетов
        - Категория для закрытых тикетов
