# Команды системы часто задаваемых вопросов (FAQ)

---

## /faq add_page
!!! info ""
    Добавить новую страницу в FAQ гильдии

    После ввода вы увидите модальное окно со следующими параметрами для ввода:

    - `Заголовок страницы` - ограничение 80 символов
    - `Краткое описание` - ограничение 200 символов
    - `Содержание страницы` - ограничение 3000 символов
    - `Ссылка на изображение` - ограничение 300 символов (опционален)

    ### Доступ
    `ADMINISTRATOR`

!!! warning "Возможные ошибки"
    - Конфигурация гильдии не найдена  
    (При возникновении введите команду `/config other setup`, можно без ввода параметров)
    - Страница с таким заголовком уже существует
    - Страница с таким кратким описанием уже существует

## /faq change_page
!!! info ""
    Изменить существующую страницу в FAQ гильдии

    После ввода вы увидите модальное окно со следующими параметрами для ввода:

    - `Новый заголовок страницы` - ограничение 100 символов (опционален)
    - `Новое краткое описание` - ограничение 200 символов (опционален)
    - `Новое содержание страницы` - ограничение 3000 символов (опционален)
    - `Ссылка на изображение` - ограничение 300 символов (опционален)

    ### Доступ
    `ADMINISTRATOR`

    ### Параметры
    - `page` - заголовок страницы, которую нужно изменить  
    Отображение вариантов для выбора с помощью автокомплита  
    Как значение передается заголовок страницы

!!! warning "Возможные ошибки"
    - Страница с выбранным названием не найдена в конфигурации
    - Конфигурация гильдии не найдена  
    (При возникновении введите команду `/config other setup`, можно без ввода параметров)

## /faq send
!!! info ""
    Отправить FAQ гильдии в текущий канал (В котором была введена команда)

    ### Доступ
    `ADMINISTRATOR`

    ### Необходимые права бота
    - `send_messages` - доступ к отправке сообщений в текущий канал для отправки сообщения с FAQ

    ### Параметры
    - `text` - текст, который будет добавлен в сообщение FAQ
    - `image_url` - вложение, которое будет прикреплено к сообщению FAQ

!!! warning "Возможные ошибки"
    - Конфигурация гильдии не найдена  
    (При возникновении введите команду `/config other setup` для создания конфигурации, можно без ввода параметров)
    - В FAQ этого сервера нет страниц для отображения (Добавьте хотя бы одну страницу)
