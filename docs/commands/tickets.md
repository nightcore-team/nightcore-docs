# Команды системы тикетов

---

## /ticketmessage
!!! info ""
    Команда для отправки панели создания тикета (Компонента `CreateTicketButton`)  
    Подробную информацию о компоненте можно найти [здесь](../components/tickets.md#createticketbutton)

    ### Доступ
    `ADMINISTRATOR`

!!! warning "Возможные ошибки"
    - Конфигурация гильдии не найдена  
    (При возникновении введите команду `/config tickets` для создания конфигурации, можно без ввода параметров)
