# Команды системы выдачи ролей

---

## /rrmessage
!!! info ""
    Команда для отправки панели создания запроса на выдачу роли (Компонента `SendRoleRequestView`)  
    Подробную информацию о компоненте можно найти [здесь](../components/role_requests.md#sendrolerequestview)

    ### Доступ
    `ADMINISTRATOR`

!!! warning "Возможные ошибки"
    - Конфигурация гильдии не найдена  
    (При возникновении введите команду `/config other setup` для создания конфигурации, можно без ввода параметров)
