# Общее про систему модерации

---

!!! tip "Логирование"
    Канал для логирования действий настраивается в [конфигурации](../commands/config.md/#config-logging), параметр `moderation`

---

!!! tip "Замена причины наказания"
    Причина наказания в командах модерации заменяется на соотвествующий пункт правила  
    Пример: ввод `1.4` заменяется на текст правила, если такой пункт существует
