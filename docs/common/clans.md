# Общее про систему кланов

---

!!! tip "Логирование"
    Канал для логирования действий настраивается в [конфигурации](../commands/config.md/#config-logging), параметр `clan`

---

!!! tip "Клановая валюта"
    Репутация - клановая валюта, получаемая каждый час за каждого из участников клана.  
    Опыт - зарабатывается за сообщения участников клана.

---

!!! tip "Участие в кланах"
    Пользователь может состоять только в 1 клане, независимо от роли
