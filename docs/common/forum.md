# Система интеграции с форумом forum.arzguard.com

---

Система создана для передачи созданных жалоб в разделе форума вашего сервера

!!! note
    - Новая жалоба закрепляется и ей присваивается префикс "На рассмотрении"
    - В определенный канал отправляется информация о жалобе с упоминанием роли:
        - `Модератор`
        - `ID Модератора`
        - `Причина`

!!! tip "Нужная информация для настройки системы"
    - `Роль` - будет пинговаться при передаче жалобы
    - `Канал` - в него будет отправляться информация о жалобе

!!! tip "Частота обновления информации"
    Получаем информацию о новых жалобах с форума каждые `60` секунд

!!! tip "Как настроить систему"
    Только напрямую связавшись с разработчиками
